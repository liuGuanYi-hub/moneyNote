## 🎯 功能使用说明

### 登录/注册
1. 打开页面，如果是新用户点击"没有账号？点击注册"
2. 填写用户名、邮箱、密码完成注册
3. 注册成功后自动登录，头像会自动分配（像素风格）

### 记账
1. 点击底部中间的"记一笔"按钮
2. 选择支出或收入类型
3. 选择分类（从后端加载）
4. 输入金额（使用数字键盘）
5. 可选添加备注
6. 点击"完成"保存

### 查看记录
1. 首页自动显示所有记账记录
2. 按日期分组显示
3. 点击记录可选择：
   - 确定 = 删除记录（进入回收站）
   - 取消 = 编辑记录（修改金额和备注）

### 统计功能
1. 点击底部"统计"标签
2. 查看：
   - 本月收支概览
   - 分类占比图表
   - 近6月收支对比

### 发现功能
1. 点击底部"发现"标签
2. 可以：
   - 查看记账小贴士
   - 获取消费分析
   - 查看预算建议
   - 查看成就系统

### 回收站
1. 进入"我的"页面
2. 点击"回收站"
3. 可以：
   - 恢复已删除的记录
   - 永久删除记录

## 🔧 技术实现

### 后端
- Spring Boot 4.0.1
- Spring Data JPA（自动建表）
- Spring Security（密码加密、JWT认证）
- MySQL数据库（money_note）
- 软删除机制（deleted字段）

### 前端
- 纯HTML + JavaScript（无框架依赖）
- Fetch API调用后端接口
- LocalStorage存储Token和用户信息
- 响应式设计，支持移动端

### API端点
- `POST /api/users/register` - 注册
- `POST /api/users/login` - 登录
- `GET /api/users/me` - 获取当前用户
- `PUT /api/users/me` - 更新用户信息
- `GET /api/categories` - 获取分类列表
- `POST /api/records` - 创建记账记录
- `GET /api/records` - 获取记录列表
- `GET /api/records/{id}` - 获取单条记录
- `PUT /api/records/{id}` - 更新记录
- `DELETE /api/records/{id}` - 删除记录（软删除）
- `GET /api/records/deleted` - 获取回收站记录
- `POST /api/records/{id}/restore` - 恢复记录
- `DELETE /api/records/{id}/permanent` - 永久删除
- `GET /api/statistics/monthly-overview` - 月度概览
- `GET /api/statistics/expense-trend` - 消费趋势
- `GET /api/statistics/category-distribution` - 分类占比
- `GET /api/statistics/income-expense-comparison` - 收支对比
